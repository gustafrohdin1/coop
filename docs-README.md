# coop

Lightweight framework för bounded agent execution. Event-driven SDK.

## Specs

Metodiken för detta framework finns i:
**[intelligent-agency/specs/agency/](https://github.com/gustafrohdin1/intelligent-agency)**

Framework-specifik usage dokumenteras i `docs/appendix-coop.md` (skapas i Fas 3).

## Struktur

```
src/coop/      ← SDK (runner, manifest, events, handlers)
handlers/      ← referens-handlers
tests/         ← tester
schema/        ← manifest JSON-schema
_discontinued/ ← legacy docs (historisk referens)
```
