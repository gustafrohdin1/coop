# Sweden API Prototype

Raw executable source package copied from `coop` branch `sweden-search-demo`
at commit `0be6225`.

This is not canonical `coop` yet.

Purpose:
- preserve the runnable prototype
- keep agent manifests, scripts, and contracts together
- make later extraction into canonical `coop` possible without rediscovery

Contents:
- `examples/sweden/agents/` — manifests for fetch, normalize, score, search
- `examples/sweden/contracts/` — schemas used by the prototype
- `examples/sweden/scripts/` — runnable Python workers

Use this folder as:
- source material
- extraction candidate
- comparison point against the study docs

Do not use this folder as:
- canonical framework API module
- proof that the general connector problem is already solved
